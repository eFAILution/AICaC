# .ai/ Directory

Structured documentation for AI coding assistants.

## Files

| File | Purpose |
|------|---------|
| `context.yaml` | Project overview, dependencies |
| `architecture.yaml` | Components and relationships |
| `workflows.yaml` | Common tasks with commands |
| `decisions.yaml` | Why things are built this way |
| `errors.yaml` | Error solutions |

## For AI Assistants

Read order:
1. `context.yaml` - Project overview
2. `workflows.yaml` - How to do things
3. `architecture.yaml` - Code structure
4. `errors.yaml` - If debugging

## For Humans

These are readable YAML files documenting the project. See `../README.md` for prose documentation.
